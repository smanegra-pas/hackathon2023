How to start the project on your VSCode locally?

1. install python on your extentions
2. Open the folder with the project inside and running the following command in a virtual environment:
	i. right click on foler and open in terminal type code .
    	ii. To make sure version type python --versian adn type python3 --version correctly
	iii. setting up the development environment type pip install -r requiremnts.txt
	iv. python manage.py makemigrations
	v. python manage.py migrate
	vi. python manage.py runserver
